Report 3: Supplier Report

Purpose: This report provides an overview of each supplier, including the number of products they supply, total orders, and the total value of orders.

Required Features:
- Joins
- Group Functions
- SQL*PLUS Report Formatting (COLUMN, TTITLE, BTITLE)

Instructions to run the report:
1. Start SQL*Plus and connect to the database.
2. Run the report_3.sql script.
